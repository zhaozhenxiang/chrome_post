POST插件介绍

  • 插件名称是什么？

    POST。

  • 视频演示

  • 如何安装该插件？

    • 在下文找到下载地址，下载文件到本地。

    • 解压文件到本地，任何时候不要删除该文件。

    • 打开Chrome开发者模式

    • 拖到文件件到Chrome插件列表中，此时Chrome会自动安装

    • 说明文档:https://blog.csdn.net/Louka/article/details/78556883

  • 该插件的作用是什么？

    最终目的是让用户能够达到分享的作用，即你浏览他人提交的URL，他人也会浏览你提交的URL。

  • 具体如何工作？

    Chrome提供插件接口，我实现接口。当你提交URL的时候会将你的URL分发给其他运行该插件的Chrome终端，该终端会打开该URL并在几秒之后打开其他的URL。当你浏览别人提交的URL时，你将获得可以累积的积分，别人浏览你的URL时会消耗你的积分。

  • 目前开发状态是什么？

    只有我一人开发。
